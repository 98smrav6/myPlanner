**Whats the purpose of this?**    

Intended use for anyone with a busy schedule and/or that wants to stay organized and keep track with their day to day work agenda. 

**Built With:**
- HTML
- JavaScript
- CSS 

**Files:**
- index.html
- style.css
- script.js
